# Heading

Description

## Examples

```javascript
<Heading asHeading={'h3'}>H3</Heading>

<Heading fontSize={'400'}>H1 font sized to 20px</Heading>

<Heading color={'green'}>A green h1</Heading>

<Heading fontWeight={'boldest'}>A 900 font weight h1</Heading>
```

## Props

- asHeading
- color
- fontSize
- fontWeight
